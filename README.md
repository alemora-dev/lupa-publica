# Lupa Pública

> Colombia public contracts and spending explorer — built on open government APIs.

Visualizes political contracts, public spending, and red flags from Colombia's SECOP II open data platform. Filter by entity, contractor, and spending category. Highlights anomalies and cross-references demographic data.

**Stack:** Vanilla JS · HTML · CSS · SECOP II API

## Run locally

```bash
# Any static server works
npx serve .
# or
python3 -m http.server 8080
```

Open `http://localhost:8080`

## Data source

[SECOP II — Sistema Electrónico de Contratación Pública](https://www.colombiacompra.gov.co/secop/secop-ii)
